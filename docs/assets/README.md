# Visual assets

This folder records the approved visual direction. Browser-ready 1200 x 900 WebP character illustrations for all 13 fighters are stored under `public/cards/`.

The shared production style is energetic, competitive and polished for ages 10+, with bold contours, painterly cel shading, tactile fruit surfaces, a dark teal arena and move-specific energy effects. Names, statistics and other typography remain controlled by the game UI rather than baked into artwork.
