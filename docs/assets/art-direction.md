# Character Artwork Direction

## Shared Production Prompt

Create a premium 2D digital collectible-card illustration for Fruit Card Arena. Match an energetic, competitive and polished game for players aged 10+ while retaining the charm of original hand-drawn fruit characters. Use bold clean ink contours, painterly cel shading, tactile fruit textures, expressive faces, dynamic action poses and a professional mobile-game finish. Place one clearly readable character in a dark navy-teal arena with dramatic overhead lighting and move-specific colored energy. Keep generous padding and a landscape-friendly composition that crops safely in a 4:3 card-art window.

Artwork must contain no card frame, UI, typography, numbers, logos, watermark, extra characters, weapons unless intrinsic to the character, photorealism or babyish proportions. Names, stats and abilities remain controlled by the game UI.

Dragon-Fruit is the style anchor for the set. Every later illustration uses it as a style-only reference.

## Web Exports

Final browser assets are 1200 x 900 WebP files at quality 86 under `public/cards/`. The source generations remain text-free so the same art can support later balance and localization changes.
