# Design Decision — Stats and Special Powers

The project owner has authorised the game designer to decide final card statistics, roles and special powers using the needs of the complete gameplay system rather than attempting to preserve every unclear handwritten number.

## Agreed approach

- Original drawings remain the source for character identity and personality.
- Visible collector statistics express each character's flavour and rarity.
- Hidden battle statistics control actual balance.
- Every character receives one simple, distinctive special ability.
- Values may be adjusted after simulation and playtesting without changing the core character concept.

This decision removes the remaining source-data blockers and allows the full roster to move into production.
