## Problems discovered

- Several handwritten values cannot be read confidently from the wide photograph.
- The original values vary greatly; gameplay balancing must not overwrite the original collector-card values without an explicit decision.
- Generated presentation artwork is useful for direction but should not be treated as the editable final card source.
